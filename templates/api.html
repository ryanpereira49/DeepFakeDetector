DeepFake Detection API


The DFD API is accessed by sending POST requests on specific URLs as described below. The base URL is http://127.0.0.1:5000/, and you obtain the full URL by appending requestâ€™s path to the base URL. You can pass two types of responses as follows.

1. Deepfake Detection request

With our API you can pass the video through POST request with the base URL and the video file.

URL = "http://127.0.0.1:5000/dfapi

files = 'Path to your file'

request.post(url=URL,file=files)

Request Response


# Field Type Description
1 accuracy String Accuracy of predection
2 date DateTime Date of Upload
3 fake_frames String No. of Fake Frames
4 name String Name of file
5 real_frames String No. of Real Frames
6 url URL Url of file
7 verdict String Detection Verdict
8 avg_frame_rate String Average Frame Rate
9 bit_rate String Video Bitrate
10 codec_name String Codec Used
11 duration String Duration Of video
12 height String Vertical Resolution of video
13 width String Horizontal Resolution of video

2. Popular Videos Data Request

A simple request with the base url combined with /data will give you the data of all the popular videos added to the public database.

URL = url_vid = "http://127.0.0.1:5000/dfapi/data

request.post(url=URL)

Request Response


# Field Type Description
1 accuracy String Accuracy of predection
2 date DateTime Date of Upload
3 fake_frames String No. of Fake Frames
4 name String Name of file
5 real_frames String No. of Real Frames
6 url URL Url of file
7 verdict String Detection Verdict
